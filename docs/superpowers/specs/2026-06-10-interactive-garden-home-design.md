# Interactive Garden Personal Homepage Design

## Design Read

Reading this as a playful personal homepage for a product engineer with creative taste, using an organic interactive garden language. The page should feel beautiful, responsive, and fun without turning into a heavy game.

## Dials

- DESIGN_VARIANCE: 8
- MOTION_INTENSITY: 6
- VISUAL_DENSITY: 4

## Direction

The homepage is a personal digital garden. Work samples are presented as seeds, skills are grouped like greenhouse zones, and the contact area closes the experience with a calm call to action.

## Structure

1. Hero Garden
   - Clear identity headline
   - Canvas based living garden background
   - Primary action to explore the page
   - Secondary action to trigger a playful seed bloom

2. Project Seeds
   - 3 to 5 project cards
   - Each card expands inline with problem, solution, and result notes
   - Cards work with click and keyboard activation

3. Skill Greenhouse
   - Four ability groups: product thinking, frontend craft, visual systems, workflow design
   - Compact and scannable, without becoming a generic tag wall

4. Contact Clearing
   - Short closing statement
   - Copy email button
   - Direct mail link

## Visual System

- Theme: single dark garden theme
- Accent: luminous lime
- Shape rule: 8px cards, pill buttons and tags
- Type: local sans stack with stronger display weight
- Motion: cursor responsive canvas, scroll reveal, card expansion, copy feedback

## Constraints

- Build as a standalone `index.html`
- No external dependencies
- No network required
- Must support mobile
- Must respect reduced motion
- Use placeholder personal content that can be replaced later
